Extract the complete recipe into the JSON structure below.

FIELD GUIDANCE:

"meta":
- "prepTime" / "cookTime" / "totalTime": In minutes. Use what the chef states. Set to null if not mentioned
- "servings": Number of servings as stated by the chef
- "difficulty": "easy" (under 30 min, basic techniques), "medium" (intermediate), "hard" (advanced techniques, long process)
- "cuisine": Type of cuisine if identifiable (e.g., "Italian", "Thai", "Mexican")

"ingredients" (include EVERY ingredient):
- "name": Ingredient name as the chef says it
- "amount": Exact quantity (e.g., "2", "1/2", "3-4")
- "unit": Measurement unit (cups, tablespoons, grams, etc.)
- "group": Group name for organization (e.g., "For the sauce", "For the dough", "For garnish")
- "notes": Preparation state or specifics (e.g., "room temperature", "finely diced", "divided")
- Include salt, pepper, oil, water — everything used

"steps" (include EVERY step):
- "number": Sequential step number
- "instruction": Clear, detailed instruction with specifics (temperatures, durations, visual cues)
- "duration": Time for this step in seconds (if applicable)
- "tips": Pro tip for this step if the chef mentions one
- "timestamp": Video timestamp in seconds
- Steps should be in EXACT order as performed
- Include visual cues: "until golden brown", "when bubbles form", "until a toothpick comes out clean"

"tips":
- Chef tips, warnings, substitutions, and storage advice
- "type": "chef_tip" (technique), "warning" (common mistake), "substitution" (alternative ingredient), "storage" (how to store)
- "text": The specific tip with detail

"substitutions":
- Alternative ingredients the chef mentions
- "original": What the recipe calls for
- "substitute": The alternative
- "notes": Impact on the final result (e.g., "for dairy-free version", "slightly different texture")

"nutrition":
- ONLY include if the chef or video explicitly mentions nutritional info
- Do NOT invent or estimate nutritional values
- Set to empty array if not mentioned

"equipment":
- All tools and equipment used (e.g., "stand mixer", "12-inch skillet", "parchment paper", "digital thermometer")
