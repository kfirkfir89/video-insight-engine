Extract the trip plan into the JSON structure below.

FIELD GUIDANCE:

"destination" / "totalDays" / "bestSeason":
- Use the EXACT destination name from the video
- totalDays: Number of days the trip covers (minimum 1)
- bestSeason: Best time to visit if mentioned, otherwise null

"days" (MINIMUM 1):
- Each day should have a clear theme and list of spots
- "city": Specific city/area for that day
- "theme": Brief theme like "Historic Old Town & Local Food"
- "dailyCost": Estimated daily cost if mentioned (include currency)
- "spots" per day:
  - "name": EXACT place name as mentioned (not abbreviated)
  - "emoji": Single emoji matching the spot type (🏛️ museum, 🍕 food, 🏖️ beach, etc.)
  - "description": What to do/see there, specific details from the video
  - "cost": Entry fee or estimated cost if mentioned
  - "duration": How long to spend (e.g., "2-3 hours")
  - "mapQuery": Google Maps searchable string — use "Place Name, City, Country" format
  - "tips": Specific tip (e.g., "Book online to skip the 2-hour line", "Go before 9am")
- "tips": Day-level tips (e.g., "Wear comfortable shoes — lots of cobblestones")

"budget":
- "total": Total estimated trip cost as a number
- "currency": Three-letter code (USD, EUR, GBP, etc.)
- "breakdown": Split by category with specific notes
  - Categories: Accommodation, Food, Transport, Activities, Shopping, Miscellaneous
  - "notes": Specific recommendations (e.g., "Airbnb in Trastevere neighborhood")
  - Only include categories with actual cost information from the video

"packingList" (MINIMUM 5):
- Items specifically relevant to this destination and activities
- "category": Essentials, Clothing, Electronics, Documents, Health
- "essential": true for must-have items, false for nice-to-have
- Include destination-specific items (sunscreen for beach trips, adapters for international, etc.)
