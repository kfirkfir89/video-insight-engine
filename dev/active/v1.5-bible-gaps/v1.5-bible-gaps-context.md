## Key Decisions

### D1: Anonymous Output Storage ‚Üí localStorage Only
- `auth-store.ts` tracks `anonymousOutputCount` in localStorage
- On signup: reset count; on login: fetch existing outputs
- Implemented: `incrementAnonymousCount()`, `shouldShowAuthPrompt()`, `resetAnonymousCount()`

### D2: Glass System ‚Üí Two-Tier (Solid + Elevated)
- `.glass` = solid `var(--card)` bg + glass border/radius (NO blur)
- `.glass-elevated` = real `backdrop-filter: blur(20px)` ‚Äî max 3-4 per screen
- `@supports` fallback for in-app browsers
- Implemented: `apps/web/src/index.css`, `apps/web/src/components/ui/glass-elevated.tsx`

### D3: Production SSR ‚Üí Vercel Rewrites to Railway
- `/s/*` requests rewritten from Vercel edge to Railway API
- Railway renders HTML via `ssr.routes.ts`
- Implemented: `vercel.json` with 60s edge cache

### D4: Shared Outputs ‚Üí Never Expire
- Even if owner downgrades to free tier
- Guard: `shareSlug: { $exists: false }` in `setExpirationForUser()`
- Implemented: `api/src/repositories/video.repository.ts`

### D5: Pro‚ÜíFree Downgrade ‚Üí Grandfather Existing
- Existing outputs keep no expiration
- Only NEW outputs after downgrade get 30d TTL (`afterDate` param)
- Implemented: `payment.service.ts:setUserTier()`

### D6: Override Timing ‚Üí Pre-Generation Only
- Override allowed AFTER detection_result SSE, BEFORE first chapter_ready
- Mid-stream override returns 409 Conflict
- Implemented: `override_state.py` + `override.py` + `stream.py`

### D7: Day-25 Warning ‚Üí Deferred to Phase 2
- TODO in payment.service.ts for SES email warning
- In-app banner at 5 days from expiry (frontend reads `expiresAt`)

### D8: OutputType Naming ‚Üí Keep Current Mapping
- `packages/types/src/index.ts` is the SINGLE source of truth
- Python mirrors documented in JSDoc comments

### D9: SwipeableView ‚Üí Deferred
- Desktop-first MVP ‚Äî touch gestures are P2

### D10: Accent Palette ‚Üí 3 Primary + 5 Rare
- Documented in `apps/web/src/index.css` above accent tokens
